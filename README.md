# Marcin Wydra — Personal Portfolio

An open-source personal portfolio website focused on performance and accessibility.  
Built with Next.js, React, TypeScript, and Tailwind CSS.

## Highlights

- **Three themes**: Light, Dark, High Contrast
- **Accessible by design**:
  - semantic HTML, ARIA where needed
  - keyboard navigation + visible focus states
  - skip-to-content link
  - reduced motion support
- **Responsive layout** (mobile → desktop)
- **Interactive globe** (work locations / projects)

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4

## Local Development

```bash
npm install
npm run dev
```
